# Covid Data Web Scraper

## Install node modules

```bash
yarn
```

## Start the application

```bash
yarn start
```

## Endpoints

Example endpoints can be found in [example.rest](https://github.com/bcsizmadia/covid-scraper/blob/main/example.rest) file.
